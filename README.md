# Portfolio 2026

Batuhan Keskinsoy icin gelistirilen, sade ve modern bir kisisel portfolio/ozgecmis sitesi.

## Teknoloji

- Next.js 16 (App Router)
- React 19
- TypeScript (strict)
- Tailwind CSS 4

## Yaklasim

- Server-first component mimarisi
- Minimum client-side JavaScript
- SEO ve performans odakli tek sayfa yapi
- Header icinden degistirilebilen dark/light mode

## Klasorleme

- `app/`: route ve layout
- `components/`: UI bilesenleri
- `lib/`: statik icerik ve yardimci yapilar
- `public/`: statik dosyalar

## Gelistirme

```bash
npm run dev
```

Tarayicida `http://localhost:3000` adresini ac.

## Build

```bash
npm run build
npm run start
```
